## 🛠️ The Data Science Toolkit

### 🧠 Machine Learning: Scikit-learn
The industry standard for ML in Python.
- [Official Site](https://scikit-learn.org/stable/index.html) | [User Guide](https://scikit-learn.org/stable/user_guide.html) | [Algorithm Examples](https://scikit-learn.org/stable/auto_examples/index.html)

### 🎨 Data Visualization: Matplotlib & Seaborn
- **Matplotlib**: The foundation of Python plotting. High control, manual setup.
- **Seaborn**: High-level interface with beautiful defaults.
- 💡 Check out the [Data Viz Project](https://datavizproject.com/) for inspiration on which chart to use!

### 🐼 Data Manipulation: Pandas & SciPy
- **Pandas**: Essential for loading (CSV, Excel, JSON, HTML) and cleaning data.
- **SciPy**: Built on NumPy, providing tools for optimization, integration, and statistics. [Docs here](https://docs.scipy.org/doc/scipy/).
